# Pelican

Provides an API to submit images to an Azure Custom Vision Object Detection model.

## Available Tags

We will want the Azure model to have images for the following tags:

```
# WET
wet-head-left
wet-head-right
wet-head-straight

bottling-right
bottling-left
bottling-straight

wet-body-right
wet-body-left

# DRY

dry-head-left
dry-head-right
dry-head-straight

dry-body-right
dry-body-left
```
